# Идентификатор

лоток (eng: tray)

# Определение

Лоток -  отсек откуда [покупатель](buyer.md) забирает оплаченый [товар](product.md)

# Пример

- Торговый автомат с лотырейными билетами 
- Торговый автомат с напитками
- Торговый автомат с едой
